# Panduan Pemasangan dan Penggunaan Aplikasi Ramadhan Countdown untuk Pengguna iOS

## Pengenalan

Aplikasi Ramadhan Countdown adalah sebuah aplikasi mudah alih yang dirancang khusus untuk membantu pengguna Islam di Malaysia dalam menyambut bulan Ramadhan yang mulia. Aplikasi ini memaparkan kiraan detik secara real-time sehingga tiba 1 Ramadhan, disertai dengan jadual waktu imsak dan berbuka untuk semua negeri di Malaysia.

---

## Cara Menggunakan Aplikasi Sekarang

### Pilihan 1: Uji Menggunakan Expo Go (Paling Mudah!)

Anda **tidak perlu menunggu** untuk menguji aplikasi Ramadhan Countdown! Anda boleh mencubanya sekarang menggunakan Expo Go.

#### Langkah-Langkah:

1. **Pasang Expo Go**
   - Buka Apple App Store di iPhone/iPad anda
   - Cari "Expo Go"
   - Tap "Get" untuk memasang

2. **Buka Aplikasi Ramadhan Countdown**
   - Buka Expo Go selepas dipasang
   - Pilih salah satu cara:
     - **Cara A:** Scan QR code di bawah menggunakan kamera anda
     - **Cara B:** Buka link ini: https://expo.dev/preview/update?message=Update+README+with+comprehensive+documentation&updateRuntimeVersion=1.0.0&createdAt=2026-01-07T00%3A57%3A35.942Z&slug=exp&projectId=c4a65723-0ed3-4dd0-9b0b-89b8bf172457&group=12cde052-68ea-4cb9-889b-5d42cc5119de

3. **Nikmati Aplikasi**
   - Aplikasi akan dimuat di Expo Go
   - Anda boleh menggunakan semua ciri-ciri aplikasi
   - Tidak perlu memasang APK atau menunggu App Store approval

#### QR Code untuk Expo Go:

Scan QR code ini dengan Expo Go untuk membuka aplikasi:

```
https://qr.expo.dev/?url=https://expo.dev/preview/update?message=Update+README+with+comprehensive+documentation&updateRuntimeVersion=1.0.0&createdAt=2026-01-07T00%3A57%3A35.942Z&slug=exp&projectId=c4a65723-0ed3-4dd0-9b0b-89b8bf172457&group=12cde052-68ea-4cb9-889b-5d42cc5119de
```

- Expo Go memerlukan sambungan internet untuk memuat aplikasi
- Setelah aplikasi dimuat, beberapa ciri mungkin memerlukan internet (seperti mengambil data jadual waktu solat terbaru)
- Mod luar talian akan berfungsi dengan data cache

## Ciri-Ciri Aplikasi

### Ciri Utama
- **Kiraan Detik Real-Time** - Memaparkan hari, jam, minit, dan saat sehingga 1 Ramadhan
- **Jadual Waktu Solat** - Jadual imsak dan berbuka untuk 70+ zon di 16 negeri Malaysia
- **Mod Gelap/Terang** - Sokongan tema gelap dan terang mengikut tetapan sistem
- **Sokongan Pelbagai Bahasa** - Tersedia dalam Bahasa Malaysia dan Bahasa Inggeris
- **Notifikasi Tolak** - Peringatan yang boleh disesuaikan 5, 10, atau 15 minit sebelum waktu imsak dan berbuka
- **Mod Luar Talian** - Aplikasi boleh digunakan tanpa sambungan internet dengan data cache 7 hari
- **Fungsi Perkongsian** - Berkongsi aplikasi ke WhatsApp, Telegram, dan platform media sosial lain
- **Kata Hikmah** - 30 kata hikmah Ramadhan yang dipaparkan secara rawak

## Soalan Lazim (FAQ)

### S: Bilakah versi iOS akan diluncurkan di App Store?
**J:** Kami sedang dalam proses pembangunan dan pengujian. Dijangka versi iOS akan diluncurkan pada Q2 2026. Sila pantau halaman GitHub untuk pengumuman rasmi.

### S: Bolehkah saya menggunakan aplikasi ini sekarang tanpa menunggu?
**J:** Ya! Anda boleh menggunakan Expo Go untuk menguji aplikasi sekarang. Lihat bahagian "Cara Menggunakan Aplikasi Sekarang" di atas.

### S: Apakah perbezaan antara Expo Go dan versi App Store?
**J:** 
- **Expo Go:** Boleh digunakan sekarang, memerlukan Expo Go app, memerlukan internet untuk memuat
- **App Store:** Akan datang kemudian, tidak perlu Expo Go, pengalaman lebih lancar, notifikasi push lebih baik

### S: Adakah kos untuk aplikasi ini?
**J:** Aplikasi ini adalah percuma untuk semua pengguna. Tiada bayaran atau langganan yang diperlukan.

### S: Adakah data saya akan disimpan secara aman?
**J:** Ya, semua data anda disimpan secara lokal di peranti anda. Tiada data yang dihantar ke pelayan luar. Privasi anda adalah prioriti utama kami.

### S: Bolehkah saya menggunakan aplikasi ini tanpa sambungan internet?
**J:** Ya, aplikasi ini mempunyai mod luar talian yang lengkap. Data jadual waktu solat akan disimpan dalam cache selama 7 hari, membolehkan anda menggunakan aplikasi tanpa sambungan internet.

### S: Bagaimanakah saya boleh memberikan maklum balas atau melaporkan pepijat?
**J:** Anda boleh melaporkan masalah atau memberikan cadangan melalui:
- GitHub Issues: https://github.com/AyamBrand/ramadhan-countdown/issues
